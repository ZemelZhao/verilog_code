# hunter模块

该模块主要实现两个功能：
1. 控制INTAN芯片；
2. 使用TYPEC与采集板进行通信；

并主要分为四个模块：
1. ADC模块 负责INTAN芯片的控制与信号采集
2. COM模块 负责与采集板进行通信；
3. CONSOLE模块 负责总体的调度和控制；
4. UTIL模块 负责其他工作
